## 概述

本项目是北京邮电大学网络工程专业2024年编译原理语法分析实验的C++实现，使用CMake + Ninja构建，支持Windows和Linux平台。

## 构建项目

1. **克隆仓库:**

   ```sh
   git clone https://github.com/xxxhois/LL1Parser.git
   cd LL1Parser
   ```

2. **构建项目:**

   运行以下命令来构建项目：

   ```sh
   cmake --preset=default
   cmake --build build
   ```

   这将根据 `CMakeLists.txt` 文件中的配置编译项目。
